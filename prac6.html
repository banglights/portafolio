La API de geolocalización HTML se utiliza para obtener la posición geográfica de un usuario. Dado que esto puede comprometer la privacidad, el puesto no está disponible a menos que el usuario lo apruebe. Es inportante por que nos brinda la oportunidad de conocer la ubicacion del usuario aunque para esto se debe pedir el concentimineto del cliente. Este tipo de apis se pude utilizar para citios de negocios ya eu en ocacione los usuarios desean saber como llegar al negocio y esta es una menera de conocer los locales cercanos.

Este ejemplo es para que el usuario encuentre su ubicación y ademas pueda visualizarla en un mapa en una pagina externa
